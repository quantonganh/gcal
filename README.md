```
$ gcal
gcal is a CLI written in Go that allows you to access Google Calendar from a command line

Usage:
  gcal [command]

Available Commands:
  event       Returns events on the specified calendar
  help        Help about any command
  list        Returns the calendars on the user's calendar list

Flags:
      --credentials string   credentials file (default is $HOME/.gcal/credentials.json)
  -h, --help                 help for gcal

Use "gcal [command] --help" for more information about a command.
subcommand is required
```
